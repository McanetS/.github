**MCAnet S.A.** es una empresa argentina con más de 35 años desarrollando soluciones de gestión para empresas medianas. Nuestro core es **IDEAFIX** sobre Linux, complementado con Node.js, PHP y PostgreSQL para integraciones modernas. Acompañamos a nuestros clientes desde el software hasta la consultoría operativa y el soporte de red.
